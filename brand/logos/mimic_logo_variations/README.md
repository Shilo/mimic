# Mimic logo variations

Generated from the provided `mimo_net.svg`, `mimo_m.svg`, and `brand.md` direction.

## Brand usage notes

- Mascot fill: Mimic Mint `#65E6B8`
- Outline / face: Midnight Ink `#10212B`
- Soft safety halo used in some wordmarks: Soft Mint `#E9FFF7`
- Wordmark font: Fredoka-style rounded wordmark converted to SVG paths, so the SVGs do **not** require a font file at runtime.
- Source font files are intentionally not included.
- Backgrounds are transparent; the preview sheet shows each logo on both white and Midnight Ink.

## Files

- `net_mimic_*`: Network Mimo + full `MIMIC` wordmark.
- `m_icon_replaces_m_*`: M-shaped Mimo replaces the first `M`, followed by `IMIC`.
- `compact_navbar_*`: Smaller navbar-oriented lockups.
- `png_previews/`: rendered PNG previews of each SVG.
- `mimic_logo_variations_light_dark_preview.png`: contact sheet.
